Musimangos is a music game where the user can guess which song is playing

## How it Works

### Getting started

  * Select Difficulty
  
  * Press Play

### Playing

  * Press "play song"
  
  * Type the name of the song into the text box
  
  * Press "guess song"

### High Scores

  * Press "load scores"
  
  * If you qualify enter name and press "add score"
